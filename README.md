# InstallParty

Getting new Ubuntu-based web developers all set up! 

## What's Included? 

- NVM
- RVM
- Latest stable releases for:
	- Ruby
    - Node
    - Rails
    - Postgres
    - MongoDB
- Sublime Text 3
- Google Chrome Stable Channel
- Atom Text Editor
- Terminator (like iTerm for Ubuntu woo)
- bash-git-prompt
